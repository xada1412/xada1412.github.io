##3.1 linux工作中常用之必备命令（一)

3.2 linux工作中常用之必备指令(二)

  1. clear命令:清屏(或者ctrl+l)

  2. who当前在本地系统上的所有用户信息
    举例:whoami；who

  3. uptime命令:查询系统信息
    举例:load average: 0.00，0.01，0.05 一分钟负载，五分钟负载，十五分钟负载

  4. w命令:查询系统信息

  5. free命令:查看系统内存
    举例:free -h；free -m

  6. wc命令:统计行
    举例:wc -1 123.txt

  7. grep命令:查找文件里符合条件的字符串
    举例:grep ‘114514’ 123.txt | wc -l

    -n:输出行数

    -w:精确匹配

    -i:忽略大小写

    -v:反向选择

  8. find命令:查询文件
    举例:find/-name -type f 123.txt

  9. uniq命令:对排序好的内容进行统计
    举例:uniq -c 123..txt | sort -n

  10. sort命令:对内容进行排序
    举例:uniq -c 123.txt | sort -n

11.

  1. netstat命令:查看网络端口的使用情况
    举例:netstat -tunlp | grep

  2. hostname命令:查看主机名

  3. ps命令:显示所有进程信息(与grep组合用，查找特定进程)
    举例:ps -ef | grep nginx；ps -aux | grep nginx

  4. kill:杀进程

  5. top命令:监控linux系统状况，比如cpu，内存的使用

  6. top命令:统计大小

举例:q键退出

基础命令3.1

  1. cd命令:切换目录
    举例:
    cd/usr/local/ ;进入指定目录
    cd.. ;返回到上一层
    cd - ;返回进入此目录之前的目录

2.ls命令:列出目录内容
举例:以列表形式展示目录ls -l；
以时间形式展示目录(倒序）ls -lt；
(正序)ls -ltr；ls[参数] [路径]；
显示隐藏路径ls -a；
显示隐藏文件的展示目录

三，pwd命令:查询所在目录

四，cat命令:查看小文件内容

举例:cat -n 123.txt（其中-n是用来排序的，显示行数）；cat ＞123.txt(新建并写文件)

五，more命令:查看大文件的内容（enter翻行 空格翻页 q退出 上下键查看上一条命令）

举例:more System.mmap_3.10.0_123.e17.x86-64

六，head命令:查看文件的前面n行

举例:head -20 system.x86-64

（默认观看文件前十行）

七，tail命令:查看文件后面n行

举例:tail -f access.log;tail -20 access.log

八，touch命令:创建一个空文件

举例:touch 123.txt

九，mkdir命令:创建目录

举例:mkdir -p/tmp/ad/ad/class

十，rmdir命令:删除目录

举例:rmdir/tmp/xd/xd/class

十一，cp命令:拷贝文件

举例:cp 123.txt class/;cp -a 123.txt class/456.txt(连同属性一起复制，可以更改名字)

十二，mv命令:移动或更名现有的文件或目录

举例:mv 123.txt 234.php(改名）;mv 789.txt/home/987.php（改名加挪到home路径下）

十三，rm命令:删除文件或目录

举例:rm 987.php(会提示是否删除）；rm -rf 456.txt(直接删除)

十四，diff命令:对比文件差异

举例:diff 123.txt 456.txt

十五，ssh命令:远程安全登录方式

举例:ssh 192.168.226.131

十六，exit命令:退出命令

十七，id命令:查看用户

举例:id root

十八，uname命令:查询主机信息

举例:uname -a

十九，ping命令:查看网络是否联通

举例:ping 192.168.1.1

二十，echo标准输出命令

举例:echo “this is echo命令”

二十一，man命令(ls –help):查看帮助文档

举例:man ls

举例:help if向

3.3 输入输出错误重定向

输入重定向

通俗的讲，输出重定向就是把要输出的的信息写入到一个文件中去
符号：<

示例：wc -l <123.txt

输出重定向

通俗的讲，输出重定向就是把要输出的信息写入到一个文件夹中去，而不是把将要输出的文件信息输出到控制台(显示屏)
> #代表覆盖写入；>> #代表追加写入

举例:cat >> 123.txt ;cat >123.txt ;ls -lrt > 123.txt ;echo ‘12345’ > 123.txt

错误重定向

举例:cat >> 123.txt ; ||| 2> /dev/null #/dev/null 无底洞

通俗的讲，错误重定向就是把错误的信息写入到一个文件中

文件描述符

0 标准输入
1 标准输出
2 标准错误输出

几个符号:
& #代表等同于的意思
&> #代表不分正确错误的意思
| #管道符
；#代表的是可以执行多条命令
&& #前面的命令执行成功的话，后面的才可以执行
前面的命令执行失败的话后面的不可以执行

|| #前面的命令执行成功的话，后面的不执行

前面的命令执行失败的话，后面的可以执行

Linux下的目录分类与作用介绍

/ :根目录,一般根目录下只存放目录,不要存放文件,也不要修改,或者是删除目录下的内容
/mnt :测试目录
/root :root用户的家目录 cd/root/;cd
/home :普通用户的家目录
/temp :临时目录(比如文件上传时)
/var :存放经常修改的数据,比如程序运行的日志文件
/boot :存放启动Linux时使用的内核文件,包括连接文件以及镜像文件 不准删除,不能删除
/etc :系统默认放置配置文件的地方
/bin :所有用户都能执行的程序
/sbin :只有root才能执行的程序
/usr :用户自己的软件都可以放到这里来
/dev :存放硬件设备的地方(/dev/cdrom)
/media :挂载光盘使用的

3.6 linux高手编辑神器vi的使用

  • vi的基本概念:(三种模式)
    命令模式
    插入模式
    底行模式

    进入插入模式:按住键盘的 i 或者 a
    进入命令模式:按住键盘的esc键
    进入底行模式:前提是得在命令模式,输入 : 进入

  • 在命令行模式中的操作:

    $ #移动到这一行的行尾
    gg #移动到文档第一行行首
    G #移动到文档最后一行行首
    x #删除内容,删除一个字符
    dd #删除游标所在的那一整行
    u #复原原来的操作
    v #选中范围内按 y 即可复制
    p #粘贴

  • 在底行模式中的操作:

    n #n为数字,光标移动到第n行
    / #寻找内容

    示例:%s/word1/word2/g #从第一行到最后一行寻找 word1 字符串,并将该字符串取代为 word2
    n1,n2s/word1/word2/g #n1 与 n2 为数字,在第 n1 行与 n2 行之间寻找 word1 这个字符串,并将该字符串取代为 word2

set nu #显示行号
set nonu #取消行号
q! #强制离开不保存
wq #离开并保存
wq! #强制离开并保存
!ls #暂时离开

3.7 linux的用户管理与组管理

简介:讲解linux用户管理与组管理

  • linux是一个多用户,多任务的操作系统
  • linux用户的分类:
    • 超级用户root:拥有最高的权限 UID:0
    • 普通用户:权限有一定限制,可以登录系统.一般可以执行/usr/local/bin或者/bin或者/usr/bin或者自己家目录的命令 UID:500-60000(centos 6) UID:1000-60000(centos 7)
    • 系统用户(伪用户):一般不会登录系统,一般情况是用来维持某个服务程序 UID:1-499(centos 6) UID:1-1000(centos 7)
  • 关于用户的相关配置文件
    • 账号信息:/etc/passwd
    • 密码信息:/etc/shadow
test :x :1000 :1000: :/home /test :/bin/bash
用户 密码占位符 UID GID 用户描述 用户家目录 登陆后使用的shell解释

/sbin/nologin #是不可登录的

  • 添加用户命令:useradd
    • -u #制定用户UID
    • -d #指定用户主目录
    • -g #指定用户所属组
    • -r #指定用户是系统用户
    • -s #用户是shell解释器
    • -M #不创建主目录
  • eg:创建一个用户HR,指定UID为1010,指定家目录为/home/HR,指定所属组为root组,指定登录shell为/bin/bash
    • useradd -u 1010 -d /home/HR -g root -s /bin/bash HR
  • 登录用户时…..
  • 删除用户命令:userdel
    • -r #连同家目录一起删除
  • 添加用户组命令:groupadd
  • 删除用户组命令:groupdel
  • 修改用户组的命令信息:usermod
    • -u #指定用户UID
    • -d #指定用户主目录
    • -g #指定用户所需组
  • 设置用户密码命令:passwd
    • passwd XD
    • echo”123456”|passwd –stdin XD
